What is Git?

Git is a version control skill used in the creation of a local repository for performing all kinds of operations, Git is used to track and managing changes in a project over a period.

Git skill is the ability to use GitHub and GitLab. GitHub and GitLab is both core hosting platform for version control collaboration. GitHub is a web-based graphic interface platform that simplifies the process of working with other people and makes it easy to collaborate on projects.

GitHub creates a centralized repository where people can work and manage projects together. GitLab is a web-based repository Git DevOps platform that enables professionals to perform all tasks in a project from planning to management and monitoring, it provides open and private repositories.

The significant difference between GitHub and GitLab is that GitHub is a collaboration platform that reviews and manages codes remotely while GitLab is majorly focused on DevOps and CI/CD.
